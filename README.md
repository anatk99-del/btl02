# מצפן זכויות לנפגעי איבה - ביטוח לאומי

פרויקט זה מרכז מידע וזכויות לנפגעי פעולות איבה, חטופים שבים ומשפחות שכולות.

## איך להפעיל ב-GitHub Pages (הוראות פשוטות)

כדי להעלות את האתר לאוויר בחינם דרך GitHub, בצעו את השלבים הבאים:

1. **יצירת המאגר (Repository):**
   - צרו Repository חדש ב-GitHub.
   - העלו אליו את הקובץ `index.html` (הקובץ היחיד שבאמת נחוץ להרצת האתר).

2. **הגדרת GitHub Pages:**
   - כנסו ללשונית **Settings** (הגדרות) בראש המאגר שלכם.
   - בתפריט הצד, לחצו על **Pages**.
   - תחת הכותרת **Build and deployment**:
     - ב-**Source** בחרו: `Deploy from a branch`.
     - ב-**Branch** בחרו: `main` (או `master`) ואת תיקיית `/ (root)`.
     - לחצו על **Save**.

3. **זהו!**
   - המתינו כדקה. בראש העמוד יופיע קישור לאתר החי שלכם (למשל: `https://username.github.io/repo-name`).

האתר בנוי כקובץ HTML עצמאי (Standalone) הכולל בתוכו את כל הקוד (React + Tailwind), כך שאין צורך בהתקנות או פקודות בניה מורכבות.
